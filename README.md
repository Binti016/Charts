
# Gauge & Pie Chart Dashboard

A minimal dashboard built with **Next.js** and **Tailwind CSS** to display a **gauge chart** and a **pie chart**.  

## Features

- Gauge Chart
- Pie Chart 
- Built with Next.js
- Styled with Tailwind CSS
